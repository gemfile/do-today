## Running
#### Clone & install

- Clone this repo `git clone git@github.com:gemfile/do-today.git`
- `cd do-today`
- run `npm install`

#### iOS

- Not available yet.

#### Android

- Run `android avd` and start an emulator
- Run `react-native run-android`
